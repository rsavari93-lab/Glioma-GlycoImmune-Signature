# Glioma Glyco-Immune Signature

## Overview

This repository contains the bioinformatics workflows and analysis scripts used in the study:

" Tumor sialylation–associated co-expression of ST6GAL1 and SIGLEC7 suggests a glyco-immune regulatory signature in low-grade glioma "

The project investigates the relationship between ST6GAL1-mediated sialylation, SIGLEC7 expression, and immune-related transcriptomic signatures in low-grade glioma (LGG).

---

## Data Sources

- TCGA-LGG
- GTEx Brain
- GEPIA2
- cBioPortal

## Analyses Included

### Expression Analysis
- ST6GAL1 expression in LGG

- ST6GAL1 vs SIGLEC7
- Spearman correlation

### Survival Analysis
- Kaplan-Meier survival curves
- Overall survival comparison

### Immune Signature Analysis
- CD8A
- PRF1
- GZMB
- NCAM1

### Functional Enrichment
- Gene Ontology (GO)
- KEGG Pathway Analysis
